
# Инструкция по установки необходимых библиотек

## Для Windows

1. **Установка Python** (если не установлен)
   - Скачайте Python 3.8+ с python.org
   - При установке отметьте "Add Python to PATH"

2. **Установка библиотек**
   ```cmd
   pip install -r requirements.txt
cmd


## Для Linux
Установка Python и зависимостей

bash
sudo apt update
sudo apt install python3 python3-pip
pip3 install -r requirements.txt


## Описание приложения

✅ **Полная интеграция с Apache Kafka**:
- Генератор отправляет данные в Kafka топик `iot-rooms`
- Продьюсер с подтверждением доставки
- Консьюмер для чтения данных

✅ **Использование Apache Flink**:
- Обработка потоков через PyFlink
- Окна обработки (tumbling windows)
- Детектор перегрева через Flink

✅ **Мастер установки**:
- Автоматическая установка всех зависимостей
- Скачивание и настройка Kafka
- Инструкции для ручной установки
- Проверка наличия Java

✅ **Управление Kafka из интерфейса**:
- Запуск/остановка Kafka broker
- Создание топиков
- Статус подключения

✅ **Кроссплатформенность**:
- Работает на Windows и Linux
- Автоматическое определение ОС
- Разные команды для разных систем

## Как использовать

1. **Первый запуск**: Нажмите "Мастер установки" для автоматической установки всех компонентов
2. **Запустите Kafka**: Кнопка "Запустить Kafka"
3. **Запустите генерацию**: "Запустить генерацию"
4. Наблюдайте за данными в реальном времени во вкладках комнат
